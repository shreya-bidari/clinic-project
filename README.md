Clinic Project (Frontend Website)

This is a simple and clean **Clinic Website** built using **HTML, CSS, and JavaScript**.  
It contains multiple pages like Home, Setup, and Allotment with a simple UI design suitable for a clinic or health-related website.

---

 Features

- Responsive homepage layout  
- Multiple HTML pages:
  - Home.html  
  - Setup.html  
  - Allotment.html  
- External CSS file for design  
- JavaScript file for basic interaction  
- Background image included  
- Simple and student-friendly project structure

---

 Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- Visual Studio Code  
- Git & GitHub
